# DECISIONES DEL SISTEMA

## 2026-01-25 – Activadores verbales KAI

Decisión:
- KAI solo se activa mediante frases explícitas ("hola kai", "oye kai").

Contexto:
- Evitar activaciones accidentales en entornos clínicos.
- Requerir intención consciente del usuario.

Estado anterior:
- Activación implícita basada en clasificación de intención.

Consecuencias:
- El frontend debe mostrar explícitamente si KAI fue activado.
- El usuario aprende el modelo mental correcto del sistema.

Estado: Vigente
